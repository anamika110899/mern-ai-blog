#  MERN AI Blog Platform  
A full-stack Blog Application built with **MongoDB, Express.js, React.js, Node.js**, and integrated **AI-powered content tools** for writers.  
Users can create, edit, delete blog posts, manage user roles, and generate AI-assisted content using modern MERN architecture.

##  Live Demo  
https://mern-ai-blog-pi.vercel.app/

The full MERN Blog App is now live on Vercel.  
Create posts, login, explore UI, and test the full flow in real time.


---

##  Features

###  User Features
-  Create, Edit & Delete blog posts  
-  User authentication (Login + Register)  
-  Roles: Admin / Writer  
-  AI-powered blog suggestions (optional)  
-  Markdown Editor support (preview mode)  
-  Upload blog banner images  

###  Admin Features
-  Manage users (writer/admin)  
-  Delete any post  
-  Monitor platform activity  
-  Block/unblock users  

###  UI/UX
-  Fast & responsive React frontend  
-  Clean dashboard with Tailwind CSS / MUI components  
-  Fully mobile responsive  

---

## Tech Stack

 **Frontend**
- React.js  
- React Router  
- Axios  
- Tailwind CSS / Material-UI  
- React Markdown Editor  

**Backend**
- Node.js  
- Express.js  
- JWT Auth  
- Multer (for images)  

 **Database**
- MongoDB Atlas  
- Mongoose ORM  

**AI Integration**
- Gemini API / OpenAI API (content suggestions)
